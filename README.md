## 🧠 What is Truth Flow?

**Truth Flow** is a visual-first system designed to help users:

- Break down complex ideas  
- Build logical flows  
- Analyze information step-by-step  
- Reach structured and meaningful conclusions  

Instead of thinking in messy blocks of text, Truth Flow lets you **think in systems**.

---

## 💡 Core Concept

> Truth is not found — it is constructed through clarity, logic, and flow.

Truth Flow transforms:
- ❌ Scattered thoughts  
- ❌ Raw data  
- ❌ Unstructured reasoning  

into:

- ✅ Clear logical chains  
- ✅ Visual flows  
- ✅ Actionable insights

## ⚙️ How to Run Locally

```bash
git clone https://github.com/Samin-Saikia/Truth.git
cd Truth
```

Then simply open:

```bash
index.html
```

Or run with a local server:

```bash
npx serve
```
